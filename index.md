# Ephemeral Delhi

## A Digital Humanities Archive of Temporary Architecture

This project documents and analyzes temporary structures in Delhi—spaces that appear, transform the city, and disappear.

Unlike permanent monuments, these structures are:
- Built quickly
- Deeply cultural
- Symbolic
- Rarely archived

## Project Aim
This project treats temporary architecture as a **cultural text**, exploring how it reshapes public space and collective memory.

## Key Focus
- Festival structures
- Wedding tents
- Protest architecture
- Commercial temporary spaces
